Compose Market provides unified access to AI models from multiple providers through a single API.

## Model Catalog

The platform maintains two catalogs:

### Curated Catalog (~2,100 models)

Models visible on the platform with:
- Verified pricing
- Validated capabilities
- Quality assurance

Access via: `GET /v1/models`

### Extended Catalog (50,000+ models)

Complete index from all providers including:
- Experimental models
- Community fine-tunes
- Specialized variants

Access via: `GET /v1/models/all`

## ModelCard Schema

Each model has a `ModelCard` with:

```typescript
interface ModelCard {
  modelId: string;           // Canonical identifier
  name: string;              // Display name
  provider: string;          // Provider ID
  taskType: string;          // Primary task
  capabilities: string[];    // Supported features
  pricing: {
    inputTokenPrice: string;
    outputTokenPrice: string;
  };
  contextWindow?: number;
  maxOutputTokens?: number;
}
```

## API Compatibility

All models use OpenAI-compatible endpoints:

- `POST /v1/chat/completions` - Text generation
- `POST /v1/images/generations` - Image generation
- `POST /v1/embeddings` - Embeddings

This allows drop-in replacement with existing OpenAI client libraries.
