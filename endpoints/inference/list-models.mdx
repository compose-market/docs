---

Compose Market provides access to an extensive catalog of AI models across multiple providers. The API follows OpenAI's `/v1/models` format for compatibility.

## Endpoints

### Standard Catalog
```
GET /v1/models
```
Returns the curated model catalog (~2,100 models) used by the platform. These models have verified pricing and validated capabilities.

### Extended Catalog
```
GET /v1/models/all
```
Returns the complete model catalog (50,000+ models) from all indexed providers. Includes experimental and community models.

## Response Schema

Each model in the response follows the `ModelCard` structure:

<ResponseField name="modelId" type="string" required>
  Canonical model identifier (e.g., `google/gemini-3-flash-preview`)
</ResponseField>

<ResponseField name="name" type="string" required>
  Human-readable display name
</ResponseField>

<ResponseField name="provider" type="string" required>
  Provider identifier: `openai`, `anthropic`, `google`, `aiml`, `huggingface`, `openrouter`, `asi`
</ResponseField>

<ResponseField name="taskType" type="string" required>
  Primary task type: `text-generation`, `image-generation`, `video-generation`, `text-to-speech`, `speech-to-text`, `embeddings`, `music-generation`
</ResponseField>

<ResponseField name="capabilities" type="array">
  Supported capabilities: `chat`, `streaming`, `tools`, `vision`, `reasoning`
</ResponseField>

<ResponseField name="pricing" type="object">
  Cost per token in USDC (6 decimals)
  
  - `inputTokenPrice` - Cost per input token
  - `outputTokenPrice` - Cost per output token
</ResponseField>

<ResponseField name="contextWindow" type="number">
  Maximum context length in tokens
</ResponseField>

<ResponseField name="maxOutputTokens" type="number">
  Maximum output length in tokens
</ResponseField>
