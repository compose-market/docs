---
title: "Health"
api: "GET https://api.compose.market/health"
description: "Check whether the API process is serving requests."
---

Returns a small JSON health payload. This route is public and does not require auth.

## Response

<ResponseField name="status" type="string">
  Health status. The live handler returns `ok`.
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO timestamp generated by the API process.
</ResponseField>

<ResponseExample>

```json Response
{
  "status": "ok",
  "timestamp": "2026-05-18T12:00:00.000Z"
}
```

</ResponseExample>
