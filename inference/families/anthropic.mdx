Compose routing is catalog-driven. A model name or docs page does not imply Anthropic support.

If an Anthropic-family model appears in the generated catalog, use its public `modelId` exactly. If it does not appear, it is not part of the hosted catalog for that build.

Use:

```http
POST /v1/models/search
```

Filter by `q` or inspect `/v1/models` for current rows.
