Use the OpenAI-compatible settings:

| Setting | Value |
| --- | --- |
| Base URL | `https://api.compose.market/external/v1` |
| Authorization | `Bearer compose-...` |
| Models endpoint | `/external/v1/models` |

There is no Hermes-specific remote config endpoint.
