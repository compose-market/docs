---

Use Compose as an OpenAI-compatible provider:

| Setting | Value |
| --- | --- |
| Base URL | `https://api.compose.market/external/v1` |
| API key | Your Compose Key |
| Models | Public IDs from `/external/v1/models` |

The API does not expose an OpenClaw-specific `/.well-known` endpoint. Use the generic external route.
