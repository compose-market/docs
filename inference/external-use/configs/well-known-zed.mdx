Configure Compose as an OpenAI-compatible provider in Zed.

| Setting | Value |
| --- | --- |
| Base URL | `https://api.compose.market/external/v1` |
| API key | Your Compose Key |
| Model | Any public ID from `/external/v1/models` |

External responses stay OpenAI-shaped, so Zed does not need to understand Compose receipts or x402 headers.

There is no Zed-specific remote config endpoint.
