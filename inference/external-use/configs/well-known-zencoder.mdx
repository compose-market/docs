Configure Zencoder as an OpenAI-compatible provider:

| Setting | Value |
| --- | --- |
| Base URL | `https://api.compose.market/external/v1` |
| API key | Your Compose Key |
| Model list | `GET https://api.compose.market/external/v1/models` |

Copy the model ID exactly from the catalog. Public aliases such as `openai/gpt-5.5` are valid model IDs.

There is no Zencoder-specific `/.well-known` endpoint.
